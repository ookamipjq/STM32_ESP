# STM32_ESP
本工程为STM32+ESP8266联网智能家居監控Demo  
Hardware:PCB1为烧录器文件，PCB6为控制板文件。  
Firmware:DHT11_ESP8266为Stm32的Keil工程，芯片为c8t6.ESP_MQTT_V1.3为Android工程
